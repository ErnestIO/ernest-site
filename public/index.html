Developer driven Orchestration

Ernest is an open source layer that lets you drive cloud infrastructure and platforms thorough a simple config file inside your application

Ernest 2.0.0 is out now!

DevOps inside your application to stop you being blocked by the slowest team

Ernest is infrastructure and platform as code defined by developers in their applications, delivered autonomously in any cloud or in house. Ernest lets you maximise the velocity of software delivery

Documentation

Compliance

Ernest has no external dependencies so you can use it to manage any compliance-affected environment

Manage

Reproduce and deploy to multiple clouds, ingest existing systems and elegantly support changes to running environments

Collaboration

A centralized tool designed to support complex collaboration and sharing across teams and organisations.

Change

Gracefully change running systems and perform orderly resizes much like a sysadmin might do manually

Ernest is open source. It's hosted, developed and maintained on GitHub.
